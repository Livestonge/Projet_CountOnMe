# Projet_CountOnMe

The IOS application CountOnMe is a simple calculator written in Swift. The project includes unit tests for the UI and for the business logic.

### Specifities
<ul>
  <li>MVC architecture</li>
  <li>Swift 5.5</li>
  <li>UIKit</li>
</ul>
